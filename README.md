# movie-site-access-orientation
本仓库用于对影视类网站进行公开访问方向性的整理说明，围绕电影与电视剧相关站点的基础访问入口与使用定位进行归纳。通过方向指引的方式，对站点名称、访问地址、站点功能定位及基础访问说明进行系统化整理，便于使用者快速判断访问目的与路径。本项目不涉及任何影视资源的存储、播放、下载或解析，仅对公开网络环境中可直接获取的访问信息进行说明性整理，所有内容来源于公开渠道，仅作访问参考用途存在。
